# BACOWR Knowledge Map

Här är en översikt över filerna i min Knowledge Base och hur du ska använda dem:

## CORE SYSTEM

- **`00_MASTER_ORCHESTRATOR.md`**: "Hjärnan". Innehåller reglerna för orkestrering, dynamisk persona-generering och hur referensartiklar syntetiseras.
- **`02_WORKFLOW.md`**: Detaljerat steg-för-steg schema för hur varje jobb ska exekveras från input till leverans.
- **`18_GPT_SETUP_GUIDE.md`**: Min egen manual.

## AGENT LOGIC (Dina sub-agenter)

- **`03_PREFLIGHT_ENGINE.md`**: Regler för research. Hur man hittar "Semantic Intersect" och analyserar publishers.
- **`04_WRITER_ENGINE.md`**: Skrivregler. Hur man hanterar struktur, ton och språk (inga AI-klichéer!).
- **`05_VALIDATOR.md`**: Kvalitetsregler. Hur LIX beräknas, lista på förbjudna ord, och hur länkar valideras.

## DATA & PROFILES

- **`06_PUBLISHER_PROFILES.md`**: Databas över svenska sajter (ton, längd, stil). Slå upp publisher här först.
- **`08_ENTITY_DATABASE.md`**: Listor på entiteter/nyckelord för olika nischer (Hälsa, Bygg, Ekonomi).
- **`09_TRUST_LINKS.md`**: Lista på auktoritativa källor (Myndigheter etc.) att använda som referenser.

## WRITING PATTERNS

- **`07_SEO_BEST_PRACTICES.md`**: Regler för ankartexter och länkplacering. "Anchor Policy".
- **`10_ARTICLE_TEMPLATES.md`**: Färdiga skelett för Guide, How-to, Listicle etc.
- **`11_SECTION_PATTERNS.md`**: Hur man bygger specifika stycken (Intro, Tips, Pros/Cons).
- **`12_INTRO_CTA_PATTERNS.md`**: Färdiga "Hooks" för inledningar och avslut.

## EXAMPLES (Ditt Facit)

- **`13_EXAMPLE_PREFLIGHT.md`**: Hur en perfekt analysrapport ser ut.
- **`14_EXAMPLE_FACIT.md`**: Hur instruktionerna till Writer ska se ut.
- **`15_EXAMPLE_ARTICLE.md`**: En perfekt slutprodukt. Jämför ditt arbete mot denna.

## API (Optional)

- **`16_API_SCHEMA.json`** & **`17_ACTION_INSTRUCTIONS.md`**: Tekniska specar om jag kopplas mot externt system.

**INSTRUKTION:** Använd dessa filer som ditt bibliotek. När du ska utföra ett steg (t.ex. skriva intro), slå upp motsvarande fil (`12_INTRO_CTA_PATTERNS`) för att hitta rätt mönster. Gissa inte.
