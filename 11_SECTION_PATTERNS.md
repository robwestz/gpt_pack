# Section Patterns

Mönster för olika sektionstyper inom artiklar.

---

## Intro-sektioner

### Hook-typer

**Statistik-hook:**

```
Var tredje svensk upplever [problem] varje år. 
Ändå är det få som vet hur enkelt det är att lösa.
```

**Fråge-hook:**

```
Har du funderat på att [handling] men inte vetat 
var du ska börja?
```

**Problem-hook:**

```
[Problem] är vanligare än du tror. Lyckligtvis 
finns det effektiva lösningar.
```

**Nyhet-hook:**

```
[Trend/utveckling] har förändrat hur vi ser på 
[ämne]. Här är vad du behöver veta.
```

### Intro-struktur

```
[Hook-mening]
[Kontext/bakgrund 1-2 meningar]
[Vad artikeln kommer täcka]
```

**Längd:** 100-150 ord

---

## Informativa sektioner

### Definition-sektion

```markdown
## Vad är [ämne]?

[Term] är [definition i en mening]. [Utvidgad 
förklaring]. [Praktiskt exempel/kontext].

Det används ofta för att [användningsområde].
```

### Förklaring-sektion

```markdown
## Hur fungerar [ämne]?

[Ämne] fungerar genom att [process]. Det innebär 
att [steg 1], sedan [steg 2], och slutligen [steg 3].

En viktig aspekt är [detalj].
```

## Värde-sektioner

### Fördelar-sektion

```markdown
## Fördelar med [ämne]

Det finns flera anledningar att överväga [ämne]:

**[Fördel 1]**
[Förklaring i 2-3 meningar].

**[Fördel 2]**
[Förklaring i 2-3 meningar].

**[Fördel 3]**
[Förklaring i 2-3 meningar].
```

### Jämförelse-sektion

```markdown
## [Alternativ A] vs [Alternativ B]

| Aspekt | [A] | [B] |
|--------|-----|-----|
| [Kriterium 1] | [Värde] | [Värde] |
| [Kriterium 2] | [Värde] | [Värde] |
| Pris | [X] kr | [Y] kr |

**Välj [A] om:** [Användarscenario]
**Välj [B] om:** [Användarscenario]
```

## Praktiska sektioner

### Steg-för-steg

```markdown
## Så [handling] - steg för steg

### Steg 1: [Rubrik]
[Instruktion]. [Eventuellt tips].

### Steg 2: [Rubrik]
[Instruktion]. [Eventuellt tips].

### Steg 3: [Rubrik]
[Instruktion]. [Eventuellt tips].
```

### Tips-sektion

```markdown
## Våra bästa tips för [mål]

1. **[Tips 1]** - [Kort förklaring]
2. **[Tips 2]** - [Kort förklaring]
3. **[Tips 3]** - [Kort förklaring]
4. **[Tips 4]** - [Kort förklaring]
5. **[Tips 5]** - [Kort förklaring]
```

### Varningar-sektion

```markdown
## Vanliga misstag att undvika

**[Misstag 1]**
[Varför det är ett problem]. Lösning: [Vad göra 
istället].

**[Misstag 2]**
[Varför det är ett problem]. Lösning: [Vad göra 
istället].
```

## Länk-kontextsektioner

### Rekommendations-sektion (för ankarlänk)

```markdown
## Vilken [produkt] ska du välja?

[Inledande kontext om valet]. När du utvärderar 
alternativ är det värt att titta på [anchor_text](url)
som erbjuder [fördel].

[Ytterligare information som stödjer rekommendationen].
```

### Resurs-sektion (för trust links)

```markdown
## Mer information

För djupare kunskap om [ämne] rekommenderar vi 
att läsa [källans](url) information om [specifikt 
ämne].

Du kan även hitta [typ av information] via 
[annan källa](url).
```

---

## Avslutnings-sektioner

### Sammanfattning

```markdown
## Sammanfattning

[Ämne] handlar i grunden om [kärna]. Genom att 
[nyckelåtgärd 1] och [nyckelåtgärd 2] kan du 
[uppnå resultat].

[Framtidsblick eller uppmuntran om relevant].
```

### Call-to-action (utan länk)

```markdown
## Nästa steg

Nu har du kunskapen du behöver för att [handling]. 
Börja med [första åtgärden] och utvärdera resultatet.

[Eventuell uppmuntran].
```

## Sektion-längder

| Sektionstyp | Ord |
|-------------|-----|
| Intro | 100-150 |
| Definition | 100-150 |
| Fördelar/tips | 150-250 |
| Steg-för-steg (totalt) | 300-500 |
| Jämförelse | 150-200 |
| Ankarlänk-sektion | 100-200 |
| Avslutning | 50-100 |
