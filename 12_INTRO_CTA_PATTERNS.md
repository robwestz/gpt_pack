# Intro and CTA Patterns

Mönster för inledningar och call-to-actions.

---

## Intro Patterns

### Pattern 1: Statistik-öppnare

```markdown
[Siffra]% av svenskar [handling/upplevelse] varje 
[tidsperiod]. Trots detta är det få som [kunskap/åtgärd]. 
I denna guide går vi igenom [vad läsaren lär sig].
```

**Exempel:**

```
67% av husägare planerar renovering inom de närmaste 
fem åren. Trots detta är det få som vet hur de ska 
välja rätt verktyg för projektet. I denna guide går 
vi igenom allt du behöver veta.
```

### Pattern 2: Problem-lösning

```markdown
[Problem] är något många kämpar med. [Konsekvens om 
olöst]. Lyckligtvis finns det [lösningstyp] som kan 
hjälpa. Här är vad du behöver veta.
```

**Exempel:**

```
Dålig sömnkvalitet är något många kämpar med. På 
sikt kan det påverka både hälsa och produktivitet. 
Lyckligtvis finns det enkla åtgärder som kan hjälpa. 
Här är vad du behöver veta.
```

### Pattern 3: Fråge-öppnare

```markdown
Har du funderat på [handling/köp/förändring] men 
inte vetat [hinder]? Du är inte ensam. [Kontext]. 
[Vad artikeln erbjuder].
```

**Exempel:**

```
Har du funderat på att byta uppvärming men inte 
vetat var du ska börja? Du är inte ensam. Många 
husägare står inför samma val. Här går vi igenom 
de vanligaste alternativen.
```

### Pattern 4: Trend-hook

```markdown
[Trend/utveckling] har förändrat hur vi [relaterat 
beteende]. [Konsekvens/möjlighet]. I denna artikel 
tittar vi närmare på [fokus].
```

**Exempel:**

```
Distansarbete har förändrat hur vi inreder våra hem. 
Allt fler behöver ett funktionellt hemmakontor. I 
denna artikel tittar vi närmare på smarta lösningar 
för små utrymmen.
```

### Pattern 5: Direkt värde

```markdown
Att [handling] kan [positivt utfall]. [Kort om hur]. 
Nedan får du [vad artikeln ger].
```

**Exempel:**

```
Att välja rätt försäkring kan spara tusentals kronor 
varje år. Nyckeln är att jämföra och förstå vad du 
verkligen behöver. Nedan får du konkreta tips för 
bättre försäkringsekonomi.
```

## Intro Don'ts

❌ **Undvik dessa öppningar:**

```
"I dagens moderna samhälle..."
"Har du någonsin undrat varför..."
"Det har blivit allt viktigare att..."
"I denna artikel kommer vi att..."
"Som vi alla vet..."
"Det finns många aspekter av..."
```

## CTA Patterns (utan länk)

### Pattern 1: Uppmuntran

```markdown
Nu har du verktygen du behöver för att [handling]. 
Börja i liten skala och utvärdera resultatet.
```

### Pattern 2: Nästa steg

```markdown
Nästa steg är att [konkret första åtgärd]. [Varför 
det är en bra start].
```

### Pattern 3: Framtidsblick

```markdown
Med rätt [kunskap/verktyg/approach] kan [positiv 
framtidsbild]. [Uppmuntrande avslutning].
```

### Pattern 4: Praktisk sammanfattning

```markdown
Sammanfattningsvis: fokusera på [nyckel 1], [nyckel 2] 
och [nyckel 3]. [Kort motivering].
```

## Länkplacerings-CTA (för ankarlänk-sektion)

### Pattern A: Rekommendation

```markdown
När det gäller [ämne] är det värt att titta på 
[anchor_text](url) som erbjuder [specifik fördel].
```

### Pattern B: Jämförelse-kontext

```markdown
Jämfört med andra alternativ på marknaden står sig 
[anchor_text](url) starkt tack vare [anledning].
```

### Pattern C: Tips-kontext

```markdown
Ett praktiskt tips är att använda [anchor_text](url) 
för att [uppnå specifikt mål].
```

### Pattern D: Expert-referens

```markdown
Många experter rekommenderar [anchor_text](url) för 
den som söker [specifik egenskap].
```

## Övergångsfraser

### För att leda till ankarlänk

- "Inom detta område finns..."
- "Ett populärt alternativ är..."
- "För den som söker kvalitet..."
- "Ett bra exempel på detta är..."

### Efter ankarlänk

- "Utöver detta finns..."
- "Oavsett vilket val du gör..."
- "Det viktigaste är att..."

---

## Avslutningsfraser

### Positiva

- "Med dessa tips är du väl rustad för..."
- "Nu har du kunskapen att..."
- "Lycka till med ditt projekt!"

### Neutrala

- "Förhoppningsvis gav detta klarhet i..."
- "Har du fler frågor kan du..."

### UNDVIK

- "Tack för att du läst"
- "Glöm inte att dela"
- "Skriv gärna en kommentar"
